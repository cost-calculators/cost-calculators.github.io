COST CALCULATORS — UPDATE & RE-UPLOAD GUIDE
===========================================
Site: https://cost-calculators.github.io/   (12 calculators + homepage)

You only edit ONE file: assets/calc-app.js  (open in Notepad / TextEdit).
At the very top are two clearly-boxed lines:

  1) LEAD EMAILS  -> var WEB3FORMS_KEY = "YOUR-WEB3FORMS-ACCESS-KEY";
  2) ANALYTICS    -> var GTM_ID = "GTM-XXXXXXX";

-------------------------------------------------------------------
A) GET LEAD EMAILS  (where your contact-form leads land)
   - Go to web3forms.com, enter the email you want leads sent to
     (e.g. nilesh.shivlab@gmail.com), click "Create Access Key".
   - A key arrives in that inbox. Paste it into line (1) above, keep quotes, save.
   - That email address = whoever you used to create the key. To CHANGE the
     destination email later, just create a new key with the new email and
     replace it on line (1). (Web3Forms routes by key, so the key IS the setting.)
   - Each lead email tells you which calculator it came from (subject line +
     a "from_page" and "page_url" field), plus the visitor's estimate range.
   - Free plan: 250 leads/month, stored 30 days. The key is safe to be public.

B) ANALYTICS (optional)
   - Google Tag Manager: tagmanager.google.com -> create a container ->
     copy the ID (GTM-XXXXXXX) -> paste into line (2), save. Done on every page.
   - Google Search Console: easiest is the "HTML file" method -- in Search
     Console choose URL-prefix https://cost-calculators.github.io/, pick the
     HTML file verification, download the googleXXXX.html file, and upload it
     into the repo (same place as index.html). OR, once GTM is live, use the
     "Google Tag Manager" verification option. Then submit sitemap.xml.

C) RE-UPLOAD
   - In your repo (cost-calculators/cost-calculators.github.io): Add file ->
     Upload files -> drag in everything from this folder -> Commit changes.
   - Wait 1-2 minutes; refresh https://cost-calculators.github.io/ (Ctrl+Shift+R).

That's it. Everything else (design, links, sitemap) is already wired for this address.
