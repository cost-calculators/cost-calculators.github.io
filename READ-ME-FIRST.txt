Live at: https://cost-calculators.github.io/

RATING SYSTEM — REBUILT FROM SCRATCH
------------------------------------
The rating code now lives INSIDE each calculator page (not in the shared JS
file), so a cached/old script can no longer break it. When you re-upload the
pages, the new rating goes live with them.

What it does now:
  * 5 clickable stars (empty until you rate).
  * Click a star -> a "Thanks for your rating!" popup appears, the star locks,
    your vote is added to the total, and the average + count update instantly.
  * One vote per calculator per device (you can't rate the same one twice).

MAKE RATINGS GLOBAL (optional, free):
  By itself a static site can only store a vote on each visitor's own device.
  To share ONE real count across all visitors, connect a free Firebase database:
    1. console.firebase.google.com -> Add project.
    2. Build -> Realtime Database -> Create -> Test mode. Copy the URL
       (https://yourproject-default-rtdb.firebaseio.com).
    3. In build (or ask me), set RATINGS_DB to that URL. Every page's <body> has
       a data-ratings-db attribute the rating reads automatically.
  Or just send me the URL and I'll bake it in.
